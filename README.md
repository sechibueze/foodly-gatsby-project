# Gatsby Web App

> Testing out building a basic food service web app with Gatsby

## Requirements

- NodeJS installed
- Git
- Gatsby
- Soem React Tip will be OK

## Local Setup

<pre>
Run the following:

`git clone https://github.com/sechibueze/foodly-gatsby-project.git`

`cd foodly-gatsby-project`

`gatsby develop`
</pre>
